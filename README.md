# Control de Gastos

Programa el cual simula un controlador de gastos usando Context API y Reducer

## Tecnologías usadas

- __React__
- __TailwindCSS__
- __TypeScript__

## Guía de instalación

```
git clone https://github.com/aidancom/control-gastos.git
cd control-gastos
npm i
npm run dev
```

__Importante__: Asegurate de tener Node.js instalado
